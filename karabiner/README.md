# Karabiner-Elements

## Setup

既存の設定は上書きされます。必要に応じて実行前にバックアップを取ってください。

```bash
cd my-pc-settings
cp karabiner/karabiner.json ~/.config/karabiner/karabiner.json
```

## Export

```bash
cp ~/.config/karabiner/karabiner.json karabiner/karabiner.json
```
