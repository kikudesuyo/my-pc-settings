# Raycast

## Export

1. Raycast で `Extensions` を開く
1. `Advanced` を開く
1. `Import / Export` から `Export` を実行する
1. 出力された内容を保存する
1. 実行時にpasswordが要求される

## Setup

1. Raycast で `Extensions` を開く
1. `Advanced` を開く
1. `Import / Export` から `Import` を実行する
1. 保存済みの内容を貼り付けて取り込む
