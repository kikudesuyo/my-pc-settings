# AltTab

## Setup

AltTab の Preferences を開き、スクリーンショットを見ながら設定します

- `Appearance.png`: 表示設定
- `Controls.png`: 操作設定
- `General.png`: 基本設定
- `Exceptions.png`: 除外アプリ設定

## Notes

- サイズは中にする
- 色は dark にする
- マウスがある場所に表示させる
- cmd+tab で表示

- AltTab、Finder、Slack, Preview は常に off にする
